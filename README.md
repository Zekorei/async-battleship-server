# Asyncronous Battleship Server

A concurrent TCP server written in C that allows multiple clients to play a Battleship-style game over a TCP connection.

The server manages player connections, game state, and message broadcasting using an event-driven architecture.

## Features
- TCP server built with POSIX sockets
- Concurrent client handling using an event loop
- Multiplayer battleship-style gameplay
- Custom text-based network protocol
- Terminal-based client

## Technologies
C, POSIX sockets, TCP/IP networking

*inspired by a networking assignment in a systems programming course*
